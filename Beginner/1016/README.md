<h2 align="center">1016 - Basic Input Output Problem</h2>
### Problem description
You are given multiple set of data and you will have to calculate the average for the class and numbers of As. 
For example,

Input: 
```
2
3
39 95 66
2
88 99
```
Output:
```
66.6666666667 1
93.5 1
```
